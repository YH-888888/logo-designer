# LOGO Designer v3-full-nocdn

一个基于 **HTML + SVG 的在线 LOGO 设计工具**，支持拖拽编辑、模板、渐变、曲线文字，并可导出高质量 SVG / PNG / JPG。  
当前版本为 **v3（稳定拖动 + 导出裁剪最终版）**。

---

## 📦 当前版本信息

- **版本号**：`v3.0.0`
- **状态**：✅ 可用 / 稳定 / 原型级产品
- **运行方式**：单 HTML 文件，浏览器直接打开
- **依赖**：❌ 无 CDN / ❌ 无第三方库（完全离线可用）
